# Product Recommendation Using Apriori Algorithm - An Analytical Approach

## Project Goal (Hypothetical)
* ABC (i.e., codename) is a company that sells food products, groceries, ingredients, and everything needed to run a practical kitchen and restaurant.
* They have an eCommerce platform for their customers to browse products, create lists, create carts and submit orders.
* The management found out that customers mostly buy the same products regularly, and their revenue growth is primarily due to signing up with new customers.
* The management wants to increase the revenue by up-selling and cross-selling products to their existing customers and the new customers.

## What does this contain.

This repository contain how you can implement analytical model to derive product recommondations using Apriori algorithm using R language.
